# Simple_Human_and_Object_Detection_System

A simple real-time Human and Object Detection System built using Python, OpenCV, and YOLOv8.

## Features

* Real-time object detection
* Human detection
* Webcam-based monitoring
* Bounding boxes with labels
* Fast and lightweight performance

## Technologies Used

* Python
* OpenCV
* YOLOv8
* NumPy

## Installation

```bash
pip install ultralytics opencv-python numpy
```

## Run the Project

```bash
python detector.py
```

## How It Works

1. Opens the webcam.
2. Captures live video frames.
3. Detects humans and objects using YOLOv8.
4. Displays detected objects with labels and confidence scores.
